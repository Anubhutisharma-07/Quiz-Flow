# 🎯 QuizFlow – Full Stack Online Quiz Application

QuizFlow is a modern full-stack online quiz platform that allows users to register, log in securely, attempt quizzes, and view their results. The application is built using Spring Boot for the backend and React for the frontend, following a RESTful architecture with JWT-based authentication.

---

## 🚀 Features

### 👤 User Authentication
- User Registration
- User Login
- JWT Authentication
- Password Encryption using BCrypt
- Protected Routes

### 📝 Quiz Management
- View Available Quizzes
- Start Quiz
- Attempt Multiple Choice Questions
- Timer During Quiz
- Automatic Quiz Submission
- Submit Quiz Responses

### 📊 Results
- Instant Quiz Score
- Percentage Calculation
- Quiz Attempt History
- Detailed Result Page

### 💻 Modern UI
- Responsive Design
- Tailwind CSS
- React Components
- Clean Dashboard
- Interactive Quiz Interface

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Anubhutisharma-07/Quiz-Flow.git
```

---

## Backend Setup

```bash
cd Backend
```

Configure MySQL database in:

```
application.properties
```

Run the Spring Boot application.

---

## Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

# 🔐 Authentication

The application uses:

- Spring Security
